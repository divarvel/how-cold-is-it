# How cold is it?

Farenheit-to-celcius converter

## Install

```
npm install
```

## Run


```
npm start
```

```
GET /celcius/0

{
  "farenheit": 32,
  "celcius": 0
}

GET /farenheit/32

{
  "farenheit": 32,
  "celcius": 0
}
```
